# E-Commerce-Backend
------------------------------------------------

## Description
------------------------------------------------
Use this application to track employees in a sql database. This app uses sequelize, mysql2, and express.

## Installation
------------------------------------------------
run npm i,
run source db/schema.sql from mysql and change seeds, then run seeds/index.js,
then run npm start
